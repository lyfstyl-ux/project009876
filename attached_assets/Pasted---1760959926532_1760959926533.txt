                          🪙 WHAT IS EVERY1.FUN? 🪙
═══════════════════════════════════════════════════════════════════════════════

Every1.fun is a Web3 platform that transforms digital content and public goods 
projects into tradeable blockchain assets (coins). We bridge traditional web 
content and Web3 innovation by allowing creators and builders to tokenize their 
work, create verifiable on-chain impact metrics, and earn sustainable income 
through trading fees.


╔═══════════════════════════════════════════════════════════════════════════╗
║  🎯 WHAT WE DO                                                            ║
╚═══════════════════════════════════════════════════════════════════════════╝

Every1.fun enables anyone to:

1. IMPORT content or project information (URL or file upload)
2. MINT it as a tradeable coin on the Base blockchain via Zora
3. TRADE these coins on an open marketplace
4. EARN passive income from trading fees (80% to creators/builders)
5. VERIFY impact with on-chain metrics for grant applications

╔═══════════════════════════════════════════════════════════════════════════╗
║  🏗️ HOW IT WORKS (TECHNICAL)                                             ║
╚═══════════════════════════════════════════════════════════════════════════╝

TECHNOLOGY STACK:
├─ Frontend: React + TypeScript + Vite
├─ Backend: Express.js + Node.js
├─ Database: PostgreSQL (Neon)
├─ Blockchain: Base (Ethereum L2)
├─ Minting: Zora Protocol
├─ Storage: IPFS via Pinata
├─ Auth: Privy (email, wallet, social)
└─ Smart Contracts: youbuidlevery1 (activity tracking)

HYBRID ON-CHAIN APPROACH:
1. User signs ONCE to deploy coin via Zora factory
2. Coin saved instantly to database for fast UX
3. Background service batches coins and records to blockchain
4. Original timestamp preserved in smart contract
5. Result: One signature + instant UX + on-chain verification
